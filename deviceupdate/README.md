# DeviceUpdate

电梯的终端升级服务。

# 远程升级格式
```
	start:

  	等待远程升级指令：	

  			上位机待命		有IP登录后等待接收指令
  			读取到指令
  					0x12 0x12 0x00	0x24 
  			开始发送数据

  	发送数据：

  			下位机待命
        delay5s
  			上位机发送数据包
  		头帧				     长度					         地址				          序号									            数据				  校验
  		俩个字节			   俩个字节	         	   四个字节（累计加）			一个字节								          2048字节			一个字节累计加
  		0x11 0x11 		 2048      			       0x8004000				    0x01（累计加，第一次为0x01）			2048字节			累计加


  	接收完成返回：
  			上位机待命
  			下位机发送指令表示已完成接收写flash，等待接收下一包
  			上位机读取
  					头帧				  序号						         校验
  					俩个字节			对应上位机发送的序号		 一个字节累计加
  					0x12 0x12		0x01 					         累计加

  	：
  	：
  	：
  	：
  	数据长度不够以实际为准;
    delay5s
  	全部发送完成等待最后一包接收完成报文后，发送结束重启指令
  			头帧			      命令			    校验
  			0x11 0x11     0x11        0x33


  	end
```
